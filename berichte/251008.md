# Grundlagen Shell

- Option, shorthand, longhand
- Argumente
- `echo` gibt was man dahinter schreibt aus
- `clear && history -c` löscht lösch Ausgabe am Bildschirm und den Verlauf der Befehle die man eingibt
- Tastenkürzel
- Filesystem
  - `ls`, `cd`, `pwd`
  - `cd ..` -> zurück zum letzten Verzeichnis
  - Arbeitsverzeichnis/working directory
- `ls -a` -> zeigt alle versteckte Dateien an

Strg + C unterbricht befehl der ausgeführt wird

# Unterschied relativer Pfad/ absoluter Pfad

- `mkdir` erstelt neuen ordner
- `touch` erstellt neue Datei
- `cd /` Wurzelverzeichnis ->
- `cd` -> wechselt zu home Verzeichnis
- `rm` löscht Dateien
- `ls ../..` zeigt bevoriges Verzeichnis
- `rmdir` -> löscht Ordner(es darf nichts in ihm vorhanden sein)
- `rm -r test/` löscht den Ordner und alles in ihm
- `rm -r /` löscht von der Wurzel aus alles (kann nur von dem mit der Berechtigung ihn zu nutzen benutzt werden)
