## Übung(Date)

Recherchiere in der manpage von date wie folgende ISO 8601 Datumsausgabe erzeugt werden kann.

```
2023-11-07T08:43:53+00:00
```
Befehl:

```sh
date --iso-8601=ns
```

## Übung (Directories und Files)

Lege mit mkdir und touch folgende Verzeichnisstruktur an:

```
./
└── abcd/
    ├── first_dir/
    │   ├── abcd.01.1.txt
    │   ├── abcd.01.2.txt
    │   └── abcd.01.3.txt
    └── second_dir/
        ├── xyz.02.1.txt
        ├── xyz.02.2.txt
        └── xyz.02.3.txt
```

Weiters:

- Stelle das Arbeitsverzeichnis auf abcd/second_dir und erstelle (ohne cd ein weiteres Mal zu verwenden):
  - ein Unterverzeichnis in_first_dir in first_dir, und darin wiederum eine Datei neu.txt
  - Endgültig sieht es dann so aus:

```
./
└── abcd/
    ├── first_dir/
    │   ├── in_first_dir/
    │   │   └── neu.txt
    │   ├── abcd.01.1.txt
    │   ├── abcd.01.2.txt
    │   └── abcd.01.3.txt
    └── second_dir/
        ├── xyz.02.1.txt
        ├── xyz.02.2.txt
        └── xyz.02.3.txt
```

Verwende den Befehl tree um (alternativ zu ls) die Ausgabe in obiger Form zu erhalten. Achtung: tree muss von REPL nachinstalliert werden, das geschieht automatisch nach der Eingabe von tree – dabei die erste dargestellte Möglichkeit auswählen.

### Ausführung:

```sh
~/workspace$ mkdir abcd
~/workspace$ cd abcd
~/workspace/abcd$ mkdir first_dir
~/workspace/abcd$ cd first_dir
~/.../abcd/first_dir$ touch abcd.01.1.txt
~/.../abcd/first_dir$ touch abcd.01.2.txt
~/.../abcd/first_dir$ touch abcd.01.3.txt
~/.../abcd/first_dir$ cd ..
~/workspace/abcd$ mkdir second_dir
~/workspace/abcd$ cd second_dir
~/.../abcd/second_dir$ touch xyz.02.1.txt
~/.../abcd/second_dir$ touch xyz.02.2.txt
~/.../abcd/second_dir$ touch xyz.02.3.txt
~/.../abcd/second_dir$ mkdir ../first_dir/in_first_dir
~/.../abcd/second_dir$ touch ../first_dir/in_first_dir/neu.txt
```

### Ausgabe:

Befehl;
```sh
~/workspace$ tree abcd
```

```sh
abcd
├── first_dir
│   ├── abcd.01.1.txt
│   ├── abcd.01.2.txt
│   ├── abcd.01.3.txt
│   └── in_first_dir
│       └── neu.txt
└── second_dir
    ├── xyz.02.1.txt
    ├── xyz.02.2.txt
    └── xyz.02.3.txt

3 directories, 7 files
```

## Übung  (Befehle – Bewegen und Kopieren)

Informiere dich über die Befehle rm, rmdir, cp und mv. Probiere die Befehle aus und dokumentiere deine Erkenntnisse im Arbeitsbericht.

- `rm` -> löscht eine Datei
- `rmdir` -> löscht Ordner(er darf keinen Inhalt haben, sonst wird er nicht gelöscht)
- `cp` -> kopiert eine Datei oder einen Ordner
- `mv` bewegt oder nennt etwas um

### Ausführung:

`rm`

```sh
~/workspace$ touch one
~/workspace$ ls
abcd  berichte  one  README.md
~/workspace$ rm one
~/workspace$ ls
abcd  berichte  README.md
```

`rmdir`

```sh
~/workspace$ mkdir sm
~/workspace$ ls
abcd  berichte  README.md  sm
~/workspace$ rmdir sm
~/workspace$ ls
abcd  berichte  README.md
```

`cp`

```sh
~/workspace$ touch red
~/workspace$ ls
abcd  berichte  README.md  red
~/workspace$ mkdir ev
~/workspace$ cp red ev
~/workspace$ ls
abcd  berichte  ev  README.md  red
~/workspace$ cd ev
~/workspace/ev$ ls
red
~/workspace/ev$ cd ..
```

`mv`

umbenennen:
```sh
~/workspace$ ls
abcd  berichte  ev  README.md  red
~/workspace$ mv red blue
~/workspace$ ls
abcd  berichte  blue  ev  README.md
```

verschieben:
```sh
~/workspace$ ls
abcd  berichte  blue  ev  README.md
~/workspace$ mv blue /home/runner/workspace/ev
~/workspace$ ls
abcd  berichte  ev  README.md
~/workspace$ cd ev
~/workspace/ev$ ls
blue  red
~/workspace/ev$ cd ..
```
