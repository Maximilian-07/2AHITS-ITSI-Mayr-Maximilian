## Übung 3.2 (more / less)
Recherchiere über die Tools more und less. Für Versuche kann die folgende Datei verwendet werden:
```
/usr/share/common-licenses/LGPL-2.1
```
Probiere:

 - Zeilen-/Seitenweise bewegen
 - Beenden
 - Nach Text suchen und von einer gefundenen Stelle zur nächsten springen

Was ist der Unterschied zwischen less und more?



Die tools `more` und `less` werden verwendet um den Inhalt von Dateien seitenweiße anzuzeigen. Sie sind ähnlich verhalten sich aber anders.

1) MORE
```
more datei.txt
```
zeigt den Inhalt von einer Datei seitenweiße an.

- Leertaste: Zeigt die nächste Seite an.

- Enter: Zeigt die nächste Zeile an.

- b: Geht eine Seite zurück.

- q: Beendet das Tool.
---

`more`ist schneller als `less` da Seiten bei `more`nur nach Bedarf geladen werden

---

2) LESS
`less` ist eine erweiterte Version von `more` und bietet eine bessere Navigation und Flexibilität

```
less datei.txt
```
zeigt den Inhalt von einer Datei an aber mit mehr Möglichkeiten zur Navigation.
- Leertaste: Geht eine Seite nach unten.

- Enter: Geht eine Zeile nach unten.

- b: Geht eine Seite nach oben.

- g: Springt zum Anfang der Datei.

- G: Springt zum Ende der Datei.

- /Suchbegriff: Sucht nach einem bestimmten Begriff im Text.

- n: Geht zum nächsten Treffer der Suche.

- N: Geht zum vorherigen Treffer der Suche.

- q: Beendet das Tool.
